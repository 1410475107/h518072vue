如何增加自己的页面？

  1. 路由配置参考文档：http://router.vuejs.org/zh-cn/essentials/getting-started.html
  2. 路由添加：src/route/index.js； 
                {
                path:'/mytest',
                component: MyTest
            }

            添加完成后这样的：
            //路由配置参考文档：http://router.vuejs.org/zh-cn/essentials/getting-started.html
            export default new Router({
                routes: [
                    {
                        path: '/',
                        name: 'HelloWorld',
                        component: HelloWorld
                    },
                    {
                        path:'/mytest',
                        component: MyTest
                    }
                ]
            })
  3. 注入组件：import MyTest from '@/components/MyTest'；
  4. 当然，你需要对应的组件；
  5. 使用rout-link添加链接：< router-link to="/mytest"> 
                    to作为属性支持动态属性值：
                < router-link :to="{name:'user',params:{userid:'123'}}">传递数据

                此时对应的路由：
                {
                    // 如果需要传递数据
                    path:'/user/:userid',
                    name:'user',
                    component: UserCenter
                }
  6. 链接地址可以传递数据： 
                    {
                    // 如果需要传递数据
                    path:'/user/:userid/:username',
                    name:'user',
                    component: UserCenter
                }
    链接加入： 
                    < router-link :to="{name:'user',params:{userid:'20181012',username:'中文'}}">传递数据
    数据的接收： 
                    {{$route.params.userid}}， {{$route.params.username}}
