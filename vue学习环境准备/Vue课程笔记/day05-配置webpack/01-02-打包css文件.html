webpack打包css文件

  1. webpack不能直接打包CSS文件。因此，想要打包CSS文件，就需要让它和某个JS文件产生关联。webpack打包时会将关联的文件一并打包。
  2. js文件可以使用require，import，url等方式关联其它文件。 我们在main.js中加入如下代码，表示引用了main.css文件。 require("../css/main.css");
  3. webpack只能处理JavaScript代码。 在打包时，发现了一个它不能处理的CSS文件。 因此我们需要用到文件对应的加载器loader。
    比如处理CSS文件就要用到两个加载器： css-loader style-loader
  4. 加载器的作用是： 将模块的源码进行转换成webpack可以识别的代码，使得webpack有能力调用外部的脚本或工具，实现对不同格式的文件的处理， 比如说分析转换less为css，转换ES6为ES5等。
  5. 加载器的使用方式一：在关联时，在路径前加上加载器名字和感叹号。
    require("style-loader!css-loader!../css/main.css");
  6. 修改代码之后再次打包：
    webpack src/js/main.js dist/js/bundle.js；
    main.js文件的代码和main.css代码都会被打包到bundle.js中。
    这样，我们只引入了一个脚本文件，却同时具备了脚本和样式。
