Less打包

  1. 开发中常常会把Less文件编译成CSS代码。 Webpack需要下载专门处理less文件的加载器----less-loader。 
    				安装：npm install less-loader --save-dev
    如果没有安装过less模块，还需要安装less： 
    				npm install less --save-dev
    使用less-loader只需要在webpack.config.js中配置一下加载器即可： 
    				module: {
			        rules: [{
			            test: /\.less$/,
			            use: [
			                'style-loader', 'css-loader', 'postcss-loader', 'less-loader'
			            ]
			        }]
				}
